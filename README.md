# OpenGL Learning Journey

After my attempts with the Gabriel Gambetta's [Graphics From Scratch](https://gabrielgambetta.com/computer-graphics-from-scratch/) and building my first tiny ray tracer and my first tiny rasterizer the next logical step was to start learning how GPU's pipeline works by picking up a graphics API.
I started with OpenGL merely because it has an excellent resource i could learn from, which is [learnopengl.com](https://learnopengl.com).

The tutorial is using C++, so i figured i should start learning C++ and i actually went through that path for a bit and it helped me to read C++ code with a bit of familiarity, but since my career is actually web development Go lang has been on my radar for quite a while, and for some reason i started learning it few days ago, then that idea crossed my mind. _Can't i just use Golang to write OpenGL code instead of C++?_

It turned out it was possible through port for GLFW and OpenGL Go bindings, and HERE WE ARE!.
My first baby triangle with OpenGL using Golang.
